<b>📖 使用说明</b>

<b>对话</b>
  直接发消息 → 发给 Claude 回答
  <code>?问题</code> → 明确发给 Claude（同上）

<b>Shell 命令</b>
  <code>!命令</code> → 在服务器上执行 shell 命令（不允许 sudo）

<b>特权 AI</b>
  <code>$指令</code> → 特权 Claude，可执行任意命令（含 sudo）、读写任意文件
  <code>$$指令</code> → 同上，但本次所有命令自动批准（免确认，每条命令会静默通知）
  <code>$clear</code> → 清空特权 AI 对话历史
  执行 shell 命令前会弹出确认消息（含三个按钮）：
    ✅ 允许一次　　📌 加入白名单　　❌ 拒绝（60秒超时自动拒绝）

<b>特权 AI 白名单</b>
  <code>$whitelist list</code> → 查看白名单
  <code>$whitelist add &lt;命令或前缀*&gt;</code> → 添加（如 <code>ls*</code> 表示前缀匹配）
  <code>$whitelist remove &lt;序号&gt;</code> → 按序号删除

<b>管理命令</b>
  <code>/help</code> — 显示此帮助
  <code>/status</code> — 查看当前 Claude backend 状态
  <code>/ctx</code> — 查看普通 AI 上下文用量明细
  <code>$ctx</code> — 查看特权 AI 上下文用量明细
  <code>/clear</code> 或 <code>!clear</code> — 清空 Claude 对话历史
  <code>/setkey &lt;API_KEY&gt;</code> — 设置 Anthropic API key，切换到 api backend
  <code>/setcli</code> — 切回 <code>claude -p</code> backend

<b>视频下载</b>
  <code>/dl 链接</code> → 下载视频（支持抖音、B站等）
  抖音：可直接粘贴分享文本，自动提取链接，无水印最高画质
  B站：4K/HDR 优先，需 cookie 解锁大会员画质
  &lt;50MB 自动上传，≥50MB 返回本地路径

<b>B站收藏夹监控</b>
  <code>/fav</code> — 查看监控状态
  <code>/fav folders</code> — 列出所有收藏夹（含 ID）
  <code>/fav list</code> — 查看监控中的收藏夹
  <code>/fav add ID</code> — 添加收藏夹到监控
  <code>/fav remove ID</code> — 移除监控
  <code>/fav download ID</code> — 下载收藏夹缺失视频
  <code>/fav download ID --force</code> — 强制重新下载全部
  <code>/fav redo BV号</code> — 强制重新下载单个视频（快速通道）
  <code>/fav clear_queue</code> — 清空下载队列（不中断当前下载）
  <code>/fav check</code> — 立即检查新视频
  <code>/fav sync</code> — 同步本地文件到 NAS
  <code>/fav queue</code> — 查看下载队列
  <code>/fav pause</code> / <code>/fav resume</code> — 暂停/恢复
  <code>/fav history</code> — 最近下载记录

<b>B站UP主监控</b>
  <code>/up</code> — 查看监控状态
  <code>/up list</code> — 查看监控中的UP主
  <code>/up add UID</code> — 添加UP主（仅通知模式）
  <code>/up add UID --download</code> — 添加UP主（自动下载模式）
  <code>/up remove UID</code> — 移除监控
  <code>/up mode UID notify/download</code> — 切换通知/下载模式
  <code>/up download UID</code> — 下载UP主缺失视频（跳过已下载）
  <code>/up download UID --force</code> — 强制重新下载全部
  <code>/up redo BV号</code> — 强制重新下载单个视频（快速通道）
  <code>/up rebuild_archive</code> — 从 NAS 重建归档索引
  <code>/up clear_queue</code> — 清空下载队列（不中断当前下载）
  <code>/up check</code> — 立即检查新视频
  <code>/up sync</code> — 同步本地文件到 NAS
  <code>/up queue</code> — 查看下载队列
  <code>/up pause</code> / <code>/up resume</code> — 暂停/恢复
  <code>/up history</code> — 最近下载记录

<b>媒体</b>
  发送图片/视频/文件 → 自动存档到服务器
  发送图片 + caption → Claude 识图（仅 api backend）
  让 Claude 发图片或视频 → Claude 会直接发送
  <code>/files</code> → 浏览归档文件（按类型分页，inline keyboard）

<b>其他</b>
  对消息添加表情反应 → Bot 回复同样的 emoji
  Claude 可渲染 LaTeX 公式（在回复中要求即可）

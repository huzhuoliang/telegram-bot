<b>📖 使用说明</b>

<b>对话</b>
  直接发消息 → 发给 Claude 回答
  <code>?问题</code> → 明确发给 Claude（同上）

<b>Shell 命令</b>
  <code>!命令</code> → 在服务器上执行 shell 命令（不允许 sudo）

<b>特权 AI</b>
  <code>$指令</code> → 特权 Claude，可执行任意命令（含 sudo）、读写任意文件
  <code>$clear</code> → 清空特权 AI 对话历史
  执行 shell 命令前会弹出确认消息（含三个按钮）：
    ✅ 允许一次　　📌 加入白名单　　❌ 拒绝（60秒超时自动拒绝）

<b>特权 AI 白名单</b>
  <code>$whitelist list</code> → 查看白名单
  <code>$whitelist add &lt;命令或前缀*&gt;</code> → 添加（如 <code>ls*</code> 表示前缀匹配）
  <code>$whitelist remove &lt;序号&gt;</code> → 按序号删除

<b>管理命令</b>
  <code>/help</code> — 显示此帮助
  <code>/status</code> — 查看当前 Claude backend 状态
  <code>/ctx</code> — 查看普通 AI 上下文用量明细
  <code>$ctx</code> — 查看特权 AI 上下文用量明细
  <code>/clear</code> 或 <code>!clear</code> — 清空 Claude 对话历史
  <code>/setkey &lt;API_KEY&gt;</code> — 设置 Anthropic API key，切换到 api backend
  <code>/setcli</code> — 切回 <code>claude -p</code> backend

<b>媒体</b>
  发送图片/视频/文件 → 自动存档到服务器
  发送图片 + caption → Claude 识图（仅 api backend）
  让 Claude 发图片或视频 → Claude 会直接发送
  <code>/files</code> → 浏览归档文件（按类型分页，inline keyboard）

<b>其他</b>
  对消息添加表情反应 → Bot 回复同样的 emoji
  Claude 可渲染 LaTeX 公式（在回复中要求即可）
